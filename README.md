\# AI SaaS User Retention \& Churn Analysis



\## Project Overview



This project analyzes user retention, churn behavior, subscription revenue, and engagement patterns for a SaaS-based product.



Using PostgreSQL and Tableau, the dataset was transformed into a product analytics dashboard that identifies key churn drivers and business opportunities.



\## Tools Used



\- PostgreSQL

\- SQL

\- Tableau

\- GitHub



\## Business Problem



SaaS companies rely on customer retention and recurring revenue for growth.



The objective of this project was to:



\- Measure churn rate

\- Analyze subscription plan performance

\- Understand user engagement patterns

\- Identify factors contributing to customer churn

\- Generate actionable business recommendations

\## Analysis Performed



\### 1. Churn Rate Analysis



Analyzed churn across subscription plans to identify retention challenges.



\### 2. Revenue Analysis



Compared revenue generated by Basic, Standard, and Premium subscription plans.



\### 3. User Engagement Analysis



Measured average weekly usage hours for retained and churned users.



\### 4. Payment Failure Analysis



Evaluated the relationship between payment failures and customer churn.



\---



\## Key Insights



\### Premium Plan Drives Revenue



Premium subscribers generated the highest revenue contribution despite similar user counts.



\### User Engagement Impacts Retention



Active users spent more time on the platform than churned users.



\- Retained Users: 13.75 hours/week

\- Churned Users: 12.25 hours/week



\### Payment Failures Increase Churn Risk



Users who churned experienced more payment failures on average.



\- Churned Users: 2.80 failures

\- Retained Users: 2.07 failures



\### High Overall Churn



The platform's churn rate of 57.32% indicates significant retention challenges.







**->Recommendations**



\- Improve payment success rates through automated reminders and payment recovery workflows.

\- Increase user engagement through personalized onboarding and feature adoption campaigns.

\- Develop retention programs targeting high-risk users.

\- Encourage upgrades to Premium plans through value-driven promotions.
